# Portfolio Website

A beautiful, modern portfolio website built with React and featuring a liquid glass design aesthetic.

## 🌟 Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Liquid Glass Effect**: Modern glassmorphic UI with blur effects and transparency
- **Smooth Animations**: Elegant transitions and animations throughout
- **Easy Customization**: Simple to update content and personalize
- **GitHub Pages Ready**: Pre-configured for easy deployment

## 📋 Sections

- **About**: Introduction and call-to-action buttons
- **Projects**: Showcase of featured projects with descriptions and links
- **Skills**: Categorized skills and proficiency levels

## 🛠️ Tech Stack

- **React** - UI library
- **CSS3** - Styling with modern features (backdrop-filter, gradients)
- **React Icons** - Icon library
- **GitHub Pages** - Hosting

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/MB-Ndhlovu/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open at `http://localhost:3000`

## ✏️ Customization

### Update Your Information

Edit the following files to personalize your portfolio:

- **`src/components/About.js`** - Update your bio, title, and links
- **`src/components/Projects.js`** - Add your projects with descriptions
- **`src/components/Skills.js`** - Update your skills and proficiencies

### Customize Colors

The main color scheme is defined in CSS files:
- Primary gradient: `#6366f1` to `#8b5cf6`
- Modify these values in component CSS files to change the theme

### Update Social Links

In `src/components/About.js`, update the social media links:
```javascript
<a href="https://github.com/YOUR-USERNAME">GitHub</a>
<a href="https://linkedin.com/in/YOUR-PROFILE">LinkedIn</a>
<a href="mailto:your.email@example.com">Email</a>
```

## 📦 Build & Deployment

### Build for Production

```bash
npm run build
```

### Deploy to GitHub Pages

```bash
npm run deploy
```

The site will be deployed to `https://MB-Ndhlovu.github.io/portfolio`

## 📱 Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
